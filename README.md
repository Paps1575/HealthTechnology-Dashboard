# HealthTechnology - Dashboard de Monitoreo Hospitalario

[cite_start]Este proyecto es una solución tecnológica desarrollada para la empresa **HealthTechnology**. [cite_start]Consiste en un Dashboard web que permite monitorear en tiempo real variables críticas de salud pública e infraestructura hospitalaria[cite: 3, 36].

## 📊 Indicadores Críticos del Dashboard
[cite_start]El sistema controla y despliega visualmente[cite: 15, 36]:
1. [cite_start]**Camas Disponibles:** Control de ocupación en tiempo real[cite: 3, 37].
2. [cite_start]**Temperatura de Refrigeradores:** Monitoreo IoT para la cadena de frío de vacunas[cite: 3, 17, 38].
3. [cite_start]**Estado del Hospital:** Estatus operativo general de la unidad médica[cite: 39].

## 🛠️ Stack Tecnológico
* [cite_start]**Frontend:** HTML5, CSS3 (Diseño responsivo y adaptativo)[cite: 15].
* [cite_start]**Control de Versiones:** Git & GitHub.

## 🔀 Estrategia de Ramas (GitFlow Simplificado)
[cite_start]Para solucionar problemas de concurrencia y desarrollo simultáneo en las APIs, el repositorio se administra bajo el siguiente esquema[cite: 9, 12, 41]:
* [cite_start]`main`: Código completamente estable y listo para producción[cite: 42].
* [cite_start]`develop`: Integración de características y pruebas del equipo de desarrollo[cite: 4, 42].
* [cite_start]`feature/dashboard-ui`: Desarrollo aislado de los componentes visuales de la interfaz[cite: 36, 42].
